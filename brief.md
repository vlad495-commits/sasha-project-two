---

## 1. О ПРОЕКТЕ

**Название:** Кофейня «Кофеин»
**Адрес:** ул. Ленина, д. 15 (вход с торца)
**Режим работы:** Пн–Вс, 8:00–22:00 (без выходных)

**Боль владельца:** Люди не знают про зал и атмосферу. Нужен сайт, который сразу говорит: «Заходите — у нас вкусно, уютно и есть где посидеть».

**Главная цель лендинга:** Превратить онлайн-посетителя в физического гостя — через бронирование столика, прокладку маршрута или звонок.

**Вторичные цели:**
- Показать атмосферу зала до первого визита (главная боль владельца)
- Сформировать желание прийти именно сейчас (акции, «тихий час»)
- Дать мгновенные ответы: адрес, часы, Wi-Fi, парковка — без поиска

**Что лендинг НЕ делает:** не является интернет-магазином, не публикует всё меню на 100 позиций.

## 2. ЦЕЛЕВАЯ АУДИТОРИЯ

Три сегмента от владельца — в порядке приоритета:

### Сегмент A: Офисные сотрудники (25–40 лет)
Хотят провести деловую встречу или пообедать вне столовой, в приятном месте.

| Параметр | Характеристика |
|---|---|
| Устройство | Мобильный (смотрит на бегу) |
| Вопрос при входе | Тихо ли? Есть ли где сесть вдвоём? |
| Триггер | Фото уютного зала + «розетки у каждого стола» |
| CTA | Забронировать столик через Telegram-бот |

### Сегмент B: Студенты и фрилансеры (18–30 лет)
Нужно место с розетками и Wi-Fi — работать за чашкой кофе.

| Параметр | Характеристика |
|---|---|
| Устройство | Мобильный + ноутбук |
| Вопрос при входе | Есть ли Wi-Fi? Розетки? Можно ли сидеть долго? |
| Триггер | Явное упоминание Wi-Fi + розеток + «без лимита по времени» |
| CTA | Посмотреть меню / прийти прямо сейчас |

### Сегмент C: Пары и компании друзей (18–40 лет)
Ищут место «посидеть-поболтать» вечером или в выходной.

| Параметр | Характеристика |
|---|---|
| Устройство | Мобильный |
| Вопрос при входе | Приятно ли там? Не шумно ли? |
| Триггер | Атмосферные фото + настольные игры + детский уголок |
| CTA | Прийти / прогуляться до нас |

> **Экспертная ремарка (UX):** 80%+ трафика — мобайл. Все дизайн-решения принимаются исходя из мобильного экрана в первую очередь.

## 4. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

| Параметр | Требование |
|---|---|
| Тип | One-page landing (якорная навигация) |
| Загрузка | < 3 сек на мобайле (LCP) — критично, 53% уходят при задержке |
| Mobile-first | Проектируется под 390px, адаптируется вверх |
| Шрифт body | min 16px |
| Зона тапа | min 48×48px для всех кнопок |
| Контраст | ≥ 4.5:1 по WCAG AA (на тёмном зелёном и на бежевом) |
| Карта | Яндекс.Карты или Google Maps — интерактивная, не скриншот |
| Аналитика | Яндекс.Метрика + Google Analytics 4 |
| SEO | Семантический HTML, meta, Open Graph |
| Телефон | Тег `<a href="tel:...">` — tap-to-call на мобайле |

### Стек
- **HTML5 + CSS3 + Vanilla JS** — без фреймворков, максимальная скорость
- **GSAP + ScrollTrigger** — scroll-анимации
- **Lenis** — smooth scroll
- **Swiper.js** — карусель отзывов
- **Google Fonts** — шрифты

## 5. ДИЗАЙН-СИСТЕМА

### 5.1 Цветовая палитра

Цвета — от владельца, взяты из реального интерьера кофейни:

```css
:root {
  /* Основные (интерьер кофейни) */
  --color-green:      #1E3320;  /* тёмно-зелёный — цвет стен, hero, footer */
  --color-green-mid:  #2A4A2E;  /* зелёный mid — alternate секции на тёмном */
  --color-beige:      #F0E6D0;  /* светлый бежевый — цвет дерева, основной фон */
  --color-beige-warm: #E8D8BE;  /* более насыщенный бежевый — карточки */

  /* Акцент */
  --color-terracotta: #C4622D;  /* терракотовый — цвет подушек, все CTA-кнопки */
  --color-terracotta-hover: #A8501F;  /* darker для hover */

  /* Текст */
  --color-text-dark:  #1A1A14;  /* почти чёрный тёплый — текст на светлом */
  --color-text-light: #F0E6D0;  /* бежевый — текст на тёмно-зелёном */
  --color-text-muted: #7A6E5E;  /* приглушённый — подписи, labels */

  /* Glassmorphism (на тёмном зелёном фоне) */
  --glass-bg:         rgba(240, 230, 208, 0.15);
  --glass-border:     rgba(240, 230, 208, 0.25);
}
```

**Принцип использования:**
- Hero, Footer, секция меню → `--color-green` (тёмный)
- Основной фон светлых секций → `--color-beige`
- Все CTA-кнопки → `--color-terracotta` (один главный CTA на секцию!)
- Текст на тёмном → `--color-text-light`
- Текст на светлом → `--color-text-dark`

### 5.2 Типографика

| Роль | Шрифт | Размер | Вес |
|---|---|---|---|
| Display (hero, заголовок «Кофеин») | Cormorant Garamond | clamp(3rem, 10vw, 8rem) | 300–400 |
| Заголовки секций | Cormorant Garamond | 2.25–3rem | 600 |
| Подзаголовки | DM Sans | 1.125rem | 400 |
| Body | DM Sans | 1rem (16px) | 400 |
| Labels / теги | DM Sans | 0.75rem | 500, UPPERCASE, letter-spacing: 0.12em |
| Цены | DM Mono | 1rem | 400 |

**Принцип:** Cormorant — для тепла и характера. DM Sans — для читаемости. Большой контраст размеров = визуальная иерархия без лишних элементов.

### 5.3 Настроение

**Ключевые слова:** тёплый · ламповый · дружелюбный · живой · домашний

**Тон текста:** Живой, разговорный, от первого лица. Не «мы предлагаем», а «заходи». Не «уникальная концепция», а «здесь помнят твой заказ со второго раза».

**Избегать:** Корпоративный язык · стоковые фото · анимационная перегрузка · тёмная тема как основная

### 5.4 Сетка и отступы

- Базовая единица: 8px
- Отступы секций: `padding: clamp(56px, 9vw, 112px) 0`
- Контейнер: max-width 1200px, padding 20px (мобайл) / 48px (десктоп)
- Сетка: CSS Grid 12 колонок, gap 24px

## 6. СТРУКТУРА СТРАНИЦЫ

### СЕКЦИЯ 0: Sticky Navigation

**Якорь:** `#nav`
**Фон:** Прозрачный над hero → `rgba(30, 51, 32, 0.95)` + `backdrop-filter: blur(12px)` после 80px скролла

```
[КОФЕИН]     [Меню]  [Акции]  [О нас]  [Контакты]    [Хочу посидеть! →]
```

**Требования:**
- Логотип «КОФЕИН» — Cormorant Garamond, текст или SVG, цвет `--color-beige`
- 4 якорных ссылки (smooth scroll), цвет `--color-text-light`
- Кнопка «Хочу посидеть!» — терракотовый, всегда видна, открывает форму бронирования
- Hamburger на мобайле (< 768px) — анимированный ×/≡

---

### СЕКЦИЯ 1: Hero

**Якорь:** `#hero`
**Фон:** `--color-green`
**Высота:** 100dvh

**Визуальный слой:**
- Полноэкранное фото зала / looping video (muted, autoplay, playsinline)
- Gradient overlay: `linear-gradient(to bottom, rgba(30,51,32,0.3), rgba(30,51,32,0.65))`
- Декоративный элемент: SVG пара из чашки или листья — лёгкая loop-анимация

**Текстовый контент:**
```
[Label — uppercase, letter-spacing]
Кофейня · ул. Ленина, 15

[Главный заголовок — Cormorant, огромный]
Кофе у нас —
это вкусно.

[Подзаголовок — DM Sans]
Кофе с собой — это быстро.
Приходи посидеть!

[CTA — главная кнопка страницы]
[ Хочу посидеть! → ]

[Micro social proof — под кнопкой]
★★★★★  4.9 · Google  ·  Пн–Вс 8:00–22:00
```

**Анимации при загрузке (stagger):**
1. Фото/фон: zoom-in scale(1.05 → 1.0) за 1.2s
2. Label: fade-in + translateY(-8px), delay 0.1s
3. Заголовок строка 1: fade-in + translateY(20px), delay 0.25s
4. Заголовок строка 2: fade-in + translateY(20px), delay 0.4s
5. Подзаголовок: fade-in, delay 0.55s
6. CTA кнопка: fade-in + scale(0.95→1), delay 0.7s
7. Social proof: fade-in, delay 0.85s

**Scroll indicator:** Анимированная стрелка ↓, исчезает после 80px скролла

> **Эксперт UX:** За 3 секунды посетитель должен понять «что за место». Адрес в label, время работы в social proof — отвечают на главные вопросы без поиска.

### СЕКЦИЯ 6: About (О нас)

**Якорь:** `#about`
**Фон:** `--color-beige`

> ⭐ Ключевая секция — передаёт атмосферу и философию кофейни. Цель: гость должен захотеть прийти, ещё не дочитав до конца.

**Структура: 2 колонки**

**Левая колонка — фото (запрос владельца):**
- Одно крупное тёплое фото: столик крупным планом — чашка кофе, ноутбук, цветок на столе
- Эффект присутствия: посетитель визуально «садится» за этот стол
- Соотношение: 4:5 или 3:4 (вертикальный кадр), тёплая экспозиция
- Опционально: второе фото снизу — бариста или общий план зала

**Правая колонка — текст:**

```
[Label — uppercase, terracotta]
О нас

[Заголовок — Cormorant, крупный]
Кофе перестаёт быть
просто напитком

[Текст — живой, тёплый, от первого лица]
«Кофеин» открылся в 2022 году — чтобы
в центре Москвы появился уютный островок,
где можно и продуктивно поработать,
и душевно провести время.

Мы не сеть и не франшиза. Здесь помнят
любимые столики и спрашивают «как обычно?» —
потому что каждый гость для нас сосед,
а не транзакция.

[Блок философии — 3 пункта, иконки + текст]

☕  Зерно от московских обжарщиков
    Работаем только с локальными обжарщиками.
    Свежая партия — каждые две недели.

🥐  Десерты из собственной мини-пекарни
    Чизкейк, круассаны, выпечка —
    готовим сами, каждый день.

👋  Ваш сосед, а не баристa
    Помним любимый столик.
    Спрашиваем «как обычно?».

[Разделитель]

[Блок удобств — иконки, компактно]

📶  Wi-Fi                    🔌  Розетки у каждого столика
🎲  Настольные игры          🎨  Детский уголок
    (шахматы, UNO, Манчкин)      (игрушки и раскраски)
🔋  Зарядные кабели (iPhone / Android) — у бариста

[Цитата про команду — serif, курсив]
«Наши бариста — настоящие кофейные гики.
Они с радостью подберут напиток
под ваше настроение.»

[Счётчики — fade-in + count-up]
[2022]          [каждые 2 нед.]    [каждый день]
Открылись       Свежая             Выпечка
в Москве        обжарка            из пекарни
```

**Анимации:**
- Фото: fade-in при входе в viewport
- Текст + абзацы: stagger fade-up (0.12s между элементами)
- Пункты философии: каждый появляется отдельно с иконкой
- Счётчики: count-up при появлении (Vanilla JS, Intersection Observer)
- Цитата: fade-in со scale(0.98 → 1) — лёгкое «всплытие»

> **Эксперт UX:** Не перегружать блок. Всё, что есть выше — это максимум. Если текст не помещается на экране — он не будет прочитан.

> **Эксперт-маркетолог:** «Каждые две недели», «собственная мини-пекарня», «зарядные кабели у бариста» — это конкретные детали, которые создают доверие. Не «мы заботимся о гостях», а вот так.

## 7. АНИМАЦИИ И ИНТЕРАКТИВ

### Принципы (из экспертного разбора конкурентов)

1. Анимации помогают, не развлекают — урок из разбора I&Me Coffee (слишком много анимаций → подвисает, рассеивается внимание)
2. Максимум 2 анимированных элемента одновременно на экране
3. Всегда `@media (prefers-reduced-motion: reduce)` — отключает всё

### Перечень анимаций

| Элемент | Что делает | Библиотека |
|---|---|---|
| Hero фон | Zoom-in (scale 1.05→1) при загрузке | CSS |
| Hero текст | Stagger fade-up (каждый элемент с задержкой) | GSAP |
| SVG пар/листья | Плавный loop вверх | CSS keyframes |
| Все секции | Fade-up при входе в viewport | GSAP ScrollTrigger |
| Карточки меню | Stagger появление (0.1s) | GSAP ScrollTrigger |
| Карточки акций | Fade-up с задержкой | GSAP ScrollTrigger |
| Маркый (Trust strip, Gallery) | Бесконечная горизонтальная прокрутка | CSS animation |
| Кнопки | Scale(1.04) + darker bg на hover | CSS transition |
| Ссылки nav | Underline draw (scaleX 0→1) | CSS pseudo |
| Карусель отзывов | Swipe + crossfade | Swiper.js |
| Sticky nav | Blur + green bg при скролле 80px | JS + CSS |
| Scroll indicator hero | Bob animation, fade out | CSS keyframes |
| Счётчики (about) | Count-up при появлении | Vanilla JS |

### Ключевые CSS-блоки

```css
/* Reveal на скролле */
.reveal {
  opacity: 0;
  transform: translateY(28px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}
.reveal.is-visible { opacity: 1; transform: none; }

/* Spring-hover для кнопок */
.btn {
  transition: transform 0.2s cubic-bezier(0.34, 1.56, 0.64, 1),
              background-color 0.2s ease;
}
.btn:hover { transform: scale(1.04); }
.btn-primary { background: var(--color-terracotta); color: var(--color-beige); }
.btn-primary:hover { background: var(--color-terracotta-hover); }

/* Glassmorphism на тёмном зелёном */
.glass {
  background: var(--glass-bg);

  -webkit-backdrop-filter: blur(12px) saturate(1.3);

  border-radius: 16px;

/* Steam / leaf loop */
```

## 10. ЧЁРНАЯ КНИГА — ЧЕГО НЕ ДЕЛАТЬ

| Запрет | Почему |
|---|---|
| Стоковые фото | Убивает доверие мгновенно — главная боль владельца: «покажи зал» |
| Меню в PDF | Не читается на мобайле, не индексируется Google |
| Поп-ап при входе | Раздражает до того, как человек увидел сайт |
| Видео со звуком автоплей | Закрывают за 1 секунду |
| Несколько главных CTA | Один лендинг — одна главная конверсия |
| Перегруженные анимации | Подвисает сайт, рассеивается внимание (урок I&Me Coffee) |
| Шрифт < 16px для body | Особенно критично для аудитории 35–45 лет |
| Некликабельный телефон | Прямые потери гостей с мобайла |
| Горизонтальный скролл | Сигнал сломанного сайта |
